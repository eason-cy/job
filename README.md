# 秋招投递记录管理系统

一个简洁的本地投递记录管理工具，数据存储在浏览器本地，无需联网。

## 快速使用

### Windows 用户
双击 `启动.bat` 文件即可自动打开浏览器访问

### Mac/Linux 用户
```bash
cd 秋招投递记录
npm install    # 首次运行需要安装依赖
npm run dev    # 启动服务
```

启动后浏览器会自动打开 http://localhost:3000

## 功能说明

- **投递记录管理** - 新增、编辑、删除投递记录
- **笔试记录** - 记录笔试日期、平台、题目等
- **面试记录** - 按轮次管理面试信息
- **数据看板** - 统计投递状态分布
- **数据备份** - 支持导出/导入 JSON 备份文件

## 数据存储

所有数据保存在浏览器的 localStorage 中，关闭浏览器后数据不会丢失。

建议定期使用"导出备份"功能保存数据文件。
